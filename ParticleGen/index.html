📖 How to use

  1. Tag the entity that will display the particles:
    /tag @e[type=player,limit=1] add particle
  2. Back up your world before running any one-command.
  3. Place a command block and paste the first one-command in.
  4. Activate it with a lever and wait for all minecarts to vanish.
  5. Repeat for each additional command in order.
  6. Add levers to the placed command blocks to activate the particle loop.
