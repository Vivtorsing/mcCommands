How to Use:

1. Tag an entity with the provided tag: /tag @e add particle

2. Make a backup of your world, as command blocks can potentially cause issues if misused.

3. The command blocks will be placed north of the initial command block position.

4. Place a single command block down and activate it with a lever to run the one-command.

5. Wait until all command block minecarts disappear before pasting and running the next one-command.

6. Once all are placed, add levers to the command blocks to activate them.
