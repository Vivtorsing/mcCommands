Command Text Display

Display the command inside a command block using a Trident or Fishing Rod!

Supported Versions:

1.21.9+

Commands by Row:

Row 1

1. [repeat] /execute as @e[type=trident,nbt={inGround:1b}] at @s unless entity @e[type=minecraft:text_display,distance=..0.5] align xyz run summon minecraft:text_display ~.5 ~ ~.5 {Tags:["commandText"],text:"Subscribe to Vivtorsing!",background:-16777216}
2. [chain] /execute as @e[type=minecraft:text_display,tag=commandText] at @s unless entity @e[type=minecraft:trident,distance=..0.5] run kill @s
3. [chain] /execute as @e[type=minecraft:text_display,tag=commandText] at @s run data modify entity @s text set from block ~ ~-1 ~ Command
4. [chain] /execute as @e[type=minecraft:text_display,tag=commandText] at @s facing entity @p feet run tp @s ~ ~ ~ ~ ~

6. [chain] /execute as @e[type=minecraft:text_display,tag=commandText] at @s unless entity @e[type=minecraft:fishing_bobber,distance=..0.5] run kill @s
7. [chain] /execute as @e[type=minecraft:text_display,tag=commandText] at @s run data modify entity @s text set from block ~ ~-1 ~ Command
